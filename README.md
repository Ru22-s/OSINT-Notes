# OSINT Notes

> ⚠️ This repository is currently under active development.  
> Some sections are incomplete, several files are still being written, and certain directories may remain empty while documentation and research are being organized.

A structured collection of Open Source Intelligence (OSINT) notes, methodologies, tools, and research related to cybersecurity, geopolitics, strategic technology, and digital investigations.

The goal of this repository is to document practical OSINT concepts in a clear and organized manner while building a long-term research and reference archive.
